第2节：语法

Gitbook 语法：标题

使用标题有两种方法，第一种是使用 "#" 开头，第二种是打完文字后在下一行加上 "==" 符号

  • 使用 "#"

    #第一层标题
##第二层标题
###第三层标题
####第四层标题
#####第五层标题

    第一层标题

    第二层标题

    第三层标题

    第四层标题

    第五层标题

  • 使用 "=="

    第一层标题
==
第二层标题
--

    第一层标题

    第二层标题

Gitbook 語法：強調

在這边会有三种强调的方式

  • 斜体，使用 "*" 或 "_" 將文字包起来

    *强调，這是斜体*

_强调，這也是斜体_

    强调，這是斜体

    强调，這也是斜体

  • 粗体，使用 "**" 或 "__" 將文字包起起來

    **強調，這是粗体**

__强调，這也是粗体__

    强调，這是粗体

    强调，這也是粗体

  • 粗斜体，使用 "**" 或 "_*" 将文字包起起來

    ***强调，這是粗斜体***

___强调，這也是粗斜体___

    强调，這是粗斜体

    强调，這也是粗斜体

Gitbook 语法：清单

清单可分为数字清单及項目清单两种，可以使用縮排进行显示，需要注意的地方为不同层级的部份需要多空一行

  • 数字清单，在前面使用 "数字. " 开头，在测试过程中发现数字清单只能到第二层，再細分下去排版会乱掉

    1. 這是数字清单 1

 1.1. 這是数字清单 1.1

2. 這是数字清单 2
3. 這是数字清单 3

    1. 这是数字清单 1

      1.1. 这是数字清单

    2. 这是数字清单 2

    3. 这是数字清单 3

  • 项目清单，在前面使用 "-" 或 "+" 开头，可以混用

    - 這是項目清單 1

 - 這是項目清單 1-1

   - 這是項目清單 1-1-1

+ 這是項目清單 2
- 這是項目清單 3
    • 這是項目清單 1
      • 這是項目清單 1-1
        • 這是項目清單 1-1-1
    • 這是項目清單 2
    • 這是項目清單 3

Gitbook 语法：链接

在使用链接部份也有两种方法可以进行

  • 使用"显示文字"进行

    [Link](http://cowmanchiang.me/gitbook)

    Link

  • 使用注解方式进行 (示范语法中以 "【 】" 取代 "[ ]" 以免发生误判)

    【Link】【1】

【1】: http://cowmanchiang.me/gitbook

    Link

Gitbook 语法：图片

插入图片的方法会跟前一节 链接 很像，差別只在於最前面多了一個 "!" 符号，或是可使用 HTML 內嵌的方式進行

  • 使用'!【图片提示文字】(图片連結)'進行 (示范语法中以 "【 】" 取代 "[ ]" 以免发生误判)

    ![Cowman](../images/77326.jpg)

  • 使用注解方式进行 (示范语法中以 "【 】" 取代 "[ ]" 以免发生误判)

    !【Cowman】【1】

【1】: ../images/77326.jpg

  • 使用 HTML 內嵌

    <img src="../images/77326.jpg" alt="Cowman" width="240" height="180" border="10" />

Gitbook 语法：代码

在代码语法显示部份也是有两种方式可以进行标示

  • 使用"8個空格"缩排

           ls -al

    ls -al

  • 使用 " '''[code language] " (以 " ''' " 取代 " ``` " 避免排版错乱) 推荐此标示方法，支持代码 highlight 显示，支持哪些语言请参考 highlightjs.org

    '''bash
ls -al
'''

    ls -al

Gitbook 语法：表格

绘制表格的方式使用 "|" 进行分隔，在使用 "-" 标示出表格Title

name | id | code
- | - | -
gitbook | 1 | nodejs
wordpress | 2 | php
name id code
gitbook 1 nodejs
wordpress 2 php

Gitbook 语法：內嵌 HTML

可以在 Markdown 中使用 HTML 语法来辅助显示的效果，例如置中、字体颜色、iframe 等

居中
<center>居中</center>
<font color="red">红色字体</font>

未居中

居中
红色字体
